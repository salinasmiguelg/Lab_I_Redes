El proyecto esta desarrollado en python 3.9 con las librerias Numpy, Matplotlib y Scipy, asegurese de tener
instaladas dichas librerias.
El codigo fuente corresponde al archivo 20.215.515-4_20.467.583-k.py 
este se puede ejecutar con un simple doble click o abriendo la consola en la carpeta en la cual se encuentra 
y escribir en ella '20.215.515-4_20.467.583-k.py' sin comillas.
al ejecutarse se solicita 'ingrese archivo de sonido:' se debe ingresar por consola el nombre con la extencion
de un archivo de audio .wav, por ejemplo G13.wav
